Static site generator

### Create

```bash
> hugo new site The-Site
> hugo new Notes/The-Note.md
```

<!--more-->

### Style

Pick theme from
https://themes.gohugo.io/

### Run

```bash
> (cd themes; git clone https://github.com/comfusion/after-dark)
> hugo server --theme=after-dark
> hugo
```
